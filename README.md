# Live Spreadsheet Dashboard Web App

A static business analytics dashboard that reads live data from the connected Google Spreadsheet. The app is read-only and does not modify spreadsheet data.

## Framework

Static HTML/CSS/JavaScript.

## Commands

```bash
npm install
npm run build
npm run preview
```

## Deploy to Vercel

- Framework Preset: Other
- Build Command: `npm run build`
- Output Directory: `dist`

See `DEPLOYMENT.md` for the full GitHub + Vercel deployment steps.
